Symmetric Multi-LiDAR Mount Configuration Optimization

This repository contains the optimization code used to study mixed discrete–continuous, symmetry-constrained multi-LiDAR placement for mobile mapping systems. The implementation includes Bayesian Optimization, Genetic Algorithm, and Particle Swarm Optimization workflows for OS1-64 dual-sensor configuration search.

What is included

The release provides three MATLAB implementations corresponding to the three optimizer families discussed in the manuscript: Bayesian Optimization, Genetic Algorithm, and Particle Swarm Optimization. Each workflow evaluates candidate LiDAR mount configurations through trajectory-based voxel simulation, overlap regularization, and angular-diversity scoring.

optimize_OS1_BO.m optimize_OS1_GA.m optimize_OS1_PSO.m

Problem setting

The optimization problem combines a discrete decision on the number of master sensors with continuous decisions on mount position and orientation. Symmetry is enforced by mirroring master sensors across the YZ plane, while optional fixed-roll constraints reduce dimensionality and reflect a practical mounting assumption for backpack-like platforms.

Bayesian Optimization

Surrogate-guided search using evaluation-efficient exploration for computationally expensive voxel-based objective functions.

Genetic Algorithm

Population-based search using crossover, mutation, and elitism to explore nonconvex multi-sensor layout configurations.

Particle Swarm Optimization

Swarm-based continuous search with convergence monitoring and optional hybrid local refinement.
